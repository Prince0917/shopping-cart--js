# 🛒 Shopping Cart App

An interactive Shopping Cart built with HTML, CSS, and JavaScript.

## ✨ Features
- Add items to the cart dynamically
- Remove items instantly
- Real-time total price calculation
- Clean and responsive UI
- Built using DOM Manipulation

## 🛠️ Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript (DOM Manipulation)

## ▶️ How to Run
Just open `index.html` in any browser — no setup needed!
